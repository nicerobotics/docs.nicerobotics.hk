# 方管塞

## 什么是方管塞?

方管塞是一种安装在方形铝管内部的小零件，它能让你轻松连接两根铝管，或者把铝管固定到板材上，而不需要焊接或复杂的连接板。

**为什么需要方管塞?**

在FRC机器人结构中，方管是最常见的基础框架材料。传统做法有两种:

- ❌ **焊接** - 不是所有队伍都有焊接条件
- ❌ **连接板** - 在狭窄空间或复杂结构中不好用

✅ **方管塞** - 简单、灵活、不需要特殊工具!

## 如何使用方管塞

!!! tip "统一标准"
    所有 NICE 方管塞都采用统一标准:
    

    - **M5螺丝孔**
    - **12.5mm孔间距**
    
    这意味着可以直接配合 NICE 铝方管使用，不需要额外打孔。

### 场景1：连接两根铝管

方管塞可以在不用任何板材的情况下直接连接两根铝管。

**安装方法:**

!!! note "方法一：单螺丝穿透"
    使用一根长M5螺丝穿过两个方管塞
    
    - ✅ 结构简单
    - ⚠️ 需要较长的螺丝

!!! note "方法二：双螺丝固定"
    将一侧孔扩大到10mm，使用两根短螺丝分别固定
    
    - ✅ 可以使用常见长度的螺丝
    - ⚠️ 需要扩孔

### 场景2：把铝管固定到板材上

方管塞也可以将铝管或CNC加工的零件与板材连接起来。
