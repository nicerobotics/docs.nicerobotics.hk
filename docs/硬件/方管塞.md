# 方管塞

在 FRC 机器人结构中，方管是最常见的基础结构件。传统做法往往依赖焊接或连接板来固定零件，但焊接条件并非每支队伍都具备，而连接板在复杂或者空间受限的场景下啊也不总是最优选择。基于这些实际问题，方管塞提供了一种更灵活的思路——让管材本身成为可靠的安装界面，从而简化装配并提升结构可用性。

## 装配

所有NICE方管塞均使用M5攻丝孔以及12.5mm孔间距，可以与NICE铝方管直接使用，不需要额外打孔。对于使用未打孔方管的队伍，可以使用NICE方管塞打孔器来精准打孔。

### 管与管连接

方管塞可以在不借助任何板材的情况下轻松连接两根铝方管。可以通过一根长M5螺丝或者将一侧孔扩大到10mm后使用短螺丝固定。

### 管与板连接

方管塞也可以将管材或者CNC零件与板材相链接。

## NICE方管塞

NICE方管塞采用6061-T6铝合金制成，整体设计在保证强度的基础上最大化减重。
