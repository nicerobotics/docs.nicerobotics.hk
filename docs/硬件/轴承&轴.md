# 轴承&轴

## 圆六角轴承

圆角六角轴承专为圆六角轴设计，能够在保证传动配合的同时提供更顺畅的转动和更好的对位效果，非常适合所有FRC常见传动结构。在承载负载时，圆六角轴承可以有效降低摩擦，使系统运行更加稳定可靠。

## 圆六角轴

轴是FRC传动中必不可少的部件，NICE圆六角轴在传统六角轴的基础上添加了中孔以及圆倒角，在减重的同时方便队伍安装。

- 采用 7075-T6 铝合金，并进行黑色电泳处理，兼顾强度与表面光滑。
- 采用中孔减重设计，极致减重的同时也方便队伍攻丝。
