# 齿轮

<div align="center">
  <img src="/assets/gear/parameter.png" width="500" />
</div>

齿轮是机械传动中最基础的零件，它能把一个轴的旋转传递到另一个轴上。更重要的是，通过搭配不同大小的齿轮，你可以调节转速和扭矩：

- 想要更大的扭矩？用小齿轮带大齿轮，转速会变慢但扭矩会增强
- 想要更快的转速？用大齿轮带小齿轮，扭矩会减弱但转速会提升

## 参数

|         名称         |              说明              |         计算公式          |
| :------------------: | :----------------------------: | :-----------------------: |
|  面宽 (Face Width)  |         齿轮齿的宽度           |         固定不变          |
| 总宽 (Overall Width) |        齿轮的整体宽度          |         固定不变          |
| 齿顶圆直径 (Outer Diameter) | 齿轮最外圈的直径       |       (齿数+2) / DP       |
| 径节 (Diametral Pitch) | 每英寸圆周上的齿数 (齿轮规格标准) |      齿数 / PD        |
| 节圆直径 (Pitch Diameter) | 两个齿轮真正接触工作时的圆直径 |      齿数 / DP         |
|   齿轮比 (Gear Ratio)   | 两个齿轮的齿数比 | 驱动齿轮齿数 / 从动齿轮齿数 |

## 齿轮比 - 转速与扭矩的转换

两个齿轮咬合在一起时，它们的大小比例 (齿轮比) 决定了输出的转速和扭矩。如果两个齿轮一样大，转速和扭矩都不会改变。

### 减速比 - 降低转速，增加扭矩

**什么时候用：** 需要更大的力量

**怎么做：** 用小齿轮 (驱动齿轮) 带动大齿轮 (从动齿轮)

**举例：** 20齿的小齿轮带动60齿的大齿轮

- 齿数比是 1:3
- 输出转速降低到原来的 1/3
- 输出扭矩提升到原来的 3倍

<div align="center">
  <img src="/assets/gear/reduction_ratio.png" width="400" />
</div>

### 加速比 - 提高转速，减少扭矩

**什么时候用：** 需要更高的最大速度

**怎么做：** 用大齿轮 (驱动齿轮) 带动小齿轮 (从动齿轮)

**举例：** 40齿的大齿轮带动20齿的小齿轮

- 齿数比是 2:1
- 输出转速提升到原来的 2倍
- 输出扭矩降低到原来的 1/2

</div>

## 旋转方向

齿轮还有一个重要作用：改变旋转方向。

- **两个齿轮：** 输入和输出方向相反 (一个顺时针，另一个就逆时针)
- **三个齿轮：** 输入和输出方向相同

<div align="center">
  <img src="/assets/gear/direction.png" width="400" />
</div>

## 什么时候使用齿轮传动

齿轮传动特别适合以下情况：

1. **距离近：** 需要传动的两个轴距离很近
2. **空间小：** 在有限的空间里需要很大的减速效果
3. **反向转：** 需要两个轴往相反方向旋转

## 中心距计算

中心距就是两个齿轮轴心之间的距离，计算公式如下：

$$
中心距=\frac{齿轮1的节圆直径}{2}+\frac{齿轮2的节圆直径}{2}+余量
$$

### 余量

在实际安装时，我们通常会在理论中心距的基础上加一点点余量 (0.03mm～0.05mm)，这样可以：

- ✅ 减少齿轮磨损
- ✅ 让齿轮转动更顺畅
- ⚠️ 但会增加一点 backlash (齿轮之间的空隙)

**推荐余量：**

- 精确配合：+0.000mm (理论值，齿轮贴合最紧)
- 余量配合：+0.03mm ～ +0.05mm (推荐使用，运行更顺畅)

## 齿间距

!!! note "重要技巧"
    只要两个齿轮的齿数总和不变，你就可以随意更换不同齿数的齿轮来调整速度和力量，而不需要改变齿轮箱的结构！

**举例：**

| 驱动齿轮 | 从动齿轮 | 齿数总和 | 齿轮比 | 中心距 |
| :------: | :------: | :------: | :----: | :----: |
|   20T    |   60T    |   80T    |  1:3   |  50.8mm (2")  |
|   32T    |   48T    |   80T    | 1:1.5  |  50.8mm (2")  |

上面两组齿轮的中心距完全相同 (都是50.8mm)，但齿轮比不同，让你可以灵活调整机器人性能。
