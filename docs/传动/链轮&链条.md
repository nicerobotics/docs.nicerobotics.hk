# 链轮&链条

## 参数

| 名称                                                 | 描述                                                         |
| :--------------------------------------------------- | :----------------------------------------------------------- |
| 节距（Pitch）                                        | 相邻链销之间的间距来区分                                     |
| 节圆直径（**P**itch **D**iameter）                   | 链条与链轮啮合工作时，链条实际绕行的那条理论圆轨迹。这个圆并不是实体结构，而是链条工作状态下形成的几何关系。 |
| 齿顶圆直径（**O**uter **D**iameter）                 | 链轮齿顶所在的最外侧直径                                     |
| 链条间隙直径（**C**hain **C**learance **D**iameter） | 链条与链轮啮合工作时，链轮在空间中实际占据的最大外轮廓尺寸   |

## 中心距计算

得益于链条每一节可以自由拆卸并且重新组装的特性，设计时可以根据需求自由设定中心距，然后通过调整链条长度以及合理的张紧使链条与链轮紧密啮合运转。需要注意的是中心距不应过小或过大，以免造成链轮干涉或无法张紧。

## 张紧

链传动需要在合理的张紧条件下才能丝滑运行。

- 链条过松：容易跳齿或脱链，磨损加剧，并引入回程间隙，影响传动稳定性与控制精度。

- 链条过紧：增加链条、轴承和传动轴负载，摩擦变大，效率下降，零部件寿命缩短。

- 张紧适中：在保持可靠啮合的同时具备适当余量，传动顺畅、磨损更小，系统效率和可靠性最佳。

### 张紧方法

- 调整链条长度：通过截链器以及Master Link可以简单有效的调整链条长度。

- 调整中心距：通过调整链轮之间的中心距可以张紧链条，但这个方法所需要的结构调整较大。

- 使用张紧器：当调整链条长度后也无法达到合适的链条紧度的话，需要使用张紧器来辅助链条张紧。

  - 惰轮（链条形成完整链轮回路的应用中）：可以布置在链环的内侧或外侧，通过改变链条路径来增加与链轮的接触面积，从而张紧链条并提升啮合稳定性。在链传动中加入惰轮并不会改变传动比。

  - 张紧器（链条不形成完整链轮回路的应用中）：通过调节中间的旋钮来张紧链条。

## 链条张紧器

链条张紧器可以通过旋转中间旋钮来实现对两头链接链条回路的张紧或放松。

- 规格：#25/#35
- 采用7075-T6铝合金制作，并且黑色阳极氧化
- 旋钮处适配M6螺栓，增加整体连接强度

使用张紧器时，先将两根固定螺栓从旋钮中拧出并且使用截链器分别与链条回路两端固定，然后将两个固定螺栓同时对准中间旋钮的两端后旋转旋钮张紧链条。

!!! warning "注意"
    链条张紧器只可以在链条不形成完整链轮回路时应用，如果张紧器转到链轮位置可能会对链轮造成损伤。
