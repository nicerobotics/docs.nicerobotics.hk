# 齿轮
<img src="/assets/gear_parameter.png" />

在机械设计中，齿轮是一种非常基础的传动元件，用来把旋转运动从一个部件传递到另一个部件。通过不同齿轮比例的搭配，还可以在机构内部实现转速与输出力矩的重新分配。转速与扭矩之间呈相反关系。提高输出力矩通常需要牺牲转速，提高输出转速通常需要牺牲力矩。

## 参数

|                 名称                 |                  描述                  |           公式            |
| :----------------------------------: | :------------------------------------: | :-----------------------: |
|      面宽（**F**ace **W**idth）      |                齿的宽度                |           恒定            |
|    总宽（**O**verall **W**idth）     |               齿轮总宽度               |           恒定            |
| 齿顶圆直径（**O**uter **D**iameter） |        齿轮齿顶所在的最外侧直径        |       (齿数+2) / DP       |
|   径节（**D**iametral **P**itch）    | 齿轮分度圆直径上每英寸长度所具有的齿数 |         齿数 / PD         |
|  节圆直径（**P**itch **D**iameter）  |         齿轮齿相互啮合的圆直径         |         齿数 / DP         |
|     齿轮比（**G**ear **R**atio）     |      指两个相互啮合齿轮的齿数之比      | 驱动齿轮齿数/从动齿轮齿数 |

## 齿轮比 - 转速与扭矩的关系

在齿轮传动中，输出的转速和扭矩取决于所使用齿轮的大小搭配。相互啮合的齿轮大小比例产生齿轮比，直接决定了转速被放大还是减小，以及扭矩的变化幅度。如果两只齿轮尺寸相同，传动前后的转速和扭矩将保持不变。

### 减速比 - 转速降低，扭矩增加

当需要降低输出转速、同时放大扭矩时，应由小齿轮去带动大齿轮。比如用一只20齿齿轮驱动一只60齿齿轮，由于两者齿数相差3倍，输出转速会相应降低到原来的1/3，而可获得约3倍的扭矩提升。

<img src="/assets/gear_reduction_ratio.png" />

### 加速比 - 转速提高，扭矩减少

当需要提高输出转速、同时减少扭矩时，应由大齿轮去带动小齿轮。比如用一只40齿齿轮驱动一只20齿齿轮，由于两者齿数相差2倍，输出转速会提高到原来的2倍，而扭矩会相应降低到原来的1/2。

## 旋转方向

除了传递功率和调整转速、扭矩之外，齿轮还可以用来改变旋转方向。相互啮合的两只齿轮，其转动方向必然相反。因此，在只包含两只齿轮的传动结构中，输出方向会与输入方向相反；而当传动链中增加到三只齿轮时，最终输出的转动方向则会与输入保持一致。

<img src="/assets/gear_direction.png"/>

## 使用场景

以下是一些更适合采用齿轮传动的典型情况：

1. 需要传递动力的部件之间距离较近。

2. 在有限空间内实现较大的减速比。

3. 需要让两个轴朝相反方向旋转。

## 中心距计算

$$
cc=\frac{PD_1}{2}+\frac{PD_2}{2}+余量
$$

### 余量

在一些设计中，会刻意在齿轮中心距上留出一定的余量，而不是采用理论上的完全贴合尺寸。这样做的好处是可以减轻齿面磨损，使齿轮箱运行更加顺畅。但需要注意的是，中心距的增加也会带来更大的回程间隙（backlash）。

常见的中心距余量参考如下：

- 精确配合：+0.000mm
- 余量配合：+0.03mm ～ +0.05mm

## 齿间距

只要两只齿轮的齿数相加总和保持不变，就可以自由调整传动比，中心距不会产生任何变化。

| 驱动齿轮 | 从动齿轮 | 齿数总和 | 齿轮比 | 齿间距 |
| :------: | :------: | :------: | :----: | :----: |
|   20T    |   60T    |   80T    |  1:3   |  相同  |
|   32T    |   48T    |   80T    | 1:1.5  |  相同  |

## 20DP 齿轮

所有NICE的20DP齿轮都拥有一致的基础参数，包括 14.5° 的压力角以及统一的9.5mm齿宽。这种标准化设计使得在不改动齿轮箱结构的前提下，就可以直接更换不同齿数的齿轮来调整传动比，大大降低了反复修改设计的成本。
