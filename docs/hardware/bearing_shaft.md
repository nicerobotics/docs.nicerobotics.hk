# 轴承&轴

## 圆六角轴承

<div align="center">
  <img src="/assets/bearing_shaft/bearing.png" />
</div>

圆六角轴承是专门为圆六角轴设计的轴承，轴承和轴需要完美配合才能顺畅转动。

**特点：**

- 转动更顺畅 - 减少摩擦阻力
- 安装更精准 - 轴承和轴自动对位，不会装歪
- 承载能力强 - 能够承受机器人运动时的各种力

!!! tip "适用场景"
    所有FRC常见的传动结构，如驱动底盘、机械臂、升降机构等。

## 圆六角轴

<div align="center">
  <img src="/assets/bearing_shaft/nice.png" />
</div>

轴是传动系统的"骨架"，负责把动力从一个地方传递到另一个地方。NICE 圆六角轴在传统六角轴的基础上做了几个重要改进：

**特点：**

- 中空设计 - 轴中间有孔，既减轻重量又方便攻丝安装零件
- 圆倒角 - 轴的六个边都做了圆角处理，让轴承转动更丝滑
- 7075-T6铝合金 - 强度高、重量轻
- 黑色电泳处理 - 表面光滑，减少摩擦

!!! tip "为什么选择中空设计？"
    - **减重** - 比实心轴轻很多，对机器人性能提升明显
    - **方便加工** - 可以直接在轴上攻丝，安装零件更简单
