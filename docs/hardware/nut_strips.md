# 螺母条

<div align="center">
  <img src="/assets/nut_strips/nice.png" width="350" />
</div>

螺母条是一种细长条状的铝合金固定件，沿着长度方向分布有连续的螺纹孔。它的作用很简单：为你的结构提供多个可用的固定点。

## 什么时候使用螺母条

螺母条特别适合以下情况：

1. **连接垂直的板件** - 两块板相互垂直时，螺母条可以快速连接
2. **把板件固定在方管上** - 螺母条放在方管内部，板件就能牢牢固定
3. **需要灵活调整** - 不需要精确对齐螺丝孔，哪里需要哪里拧

## 如何使用

<div align="center">
  <img src="/assets/nut_strips/board_and_tube.png" width="500" />
</div>

安装非常简单：

1. 将螺母条放置在板材背面或型材内部
2. 直接在需要的位置拧螺丝固定
3. 完成！

!!! tip "重要优势"
    - **不需要精确对齐** - 螺母条上有连续的螺纹孔，哪里需要哪里拧
    - **可以自由裁剪** - 根据实际需求裁成合适长度
    - **安装效率高** - 省去了对齐单个螺母的麻烦

## NICE 螺母条

**特点：**

- 6061-T6 铝合金，黑色阳极氧化处理
- M5螺纹孔，12.5mm孔距，可以与NICE铝方管配合使用
- 每根长度200mm，可以根据需求自由裁剪
